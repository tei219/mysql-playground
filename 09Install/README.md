# OS にインストール
## なにするだ
各OS毎にMySQLのインストールを試します

## シナリオ
* ディストリビューション毎のパッケージ管理ツールでMySQLをインストールします
* ディストリビューション毎用のバイナリパックでMySQLをインストールします
* ソースからMySQLをインストールします
